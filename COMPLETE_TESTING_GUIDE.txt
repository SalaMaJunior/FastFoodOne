🧪 How to Test Everything for FREE
==================================

1. 🏗️ Backend Testing (Local Development)
==========================================

Prerequisites:
- Install Node.js 18+
- Install MongoDB Community Edition (Free)
- Install Redis (Free)
- Install Docker Desktop (Free)

Step 1: Start Backend Services
------------------------------
# Navigate to backend
cd D:\FastBite-Complete\backend

# Install dependencies
npm install

# Start all services with Docker
cd ..\infrastructure
docker-compose up -d

# Or start individually for development
cd ..\backend
npm run dev

Step 2: Test API Endpoints
--------------------------
# API Gateway will be available at:
http://localhost:3000

# Test health check
curl http://localhost:3000/health

# Test API documentation
http://localhost:3000/docs

2. 📱 Mobile App Testing (React Native)
=======================================

Prerequisites:
- Install React Native CLI
  npm install -g @react-native-community/cli

- Install Android Studio (Free)
- Install Xcode (Mac only, for iOS)
- Install Expo CLI
  npm install -g @expo/cli

Step 1: Start Customer App
--------------------------
cd D:\FastBite-Complete\mobile-app\customer-app

# Install dependencies
npm install

# Start Metro bundler
npm start

# Run on Android
npm run android

# Run on iOS (Mac only)
npm run ios

Step 2: Test Mobile Features
-----------------------------
✅ Authentication - Login/Register
✅ AI Search - Voice search, smart recommendations
✅ AR Menu - Point camera at food
✅ Biometric Auth - Touch ID/Face ID
✅ Real-time Tracking - Order tracking
✅ Meal Planning - AI-powered meal plans
